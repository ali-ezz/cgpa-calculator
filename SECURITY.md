# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository, please open an issue with the label `security` or contact the repository owner directly.

## What to include

- A clear description of the vulnerability.
- Steps to reproduce.
- Affected files and components.
- Any mitigation or workaround information.

## Response

Security reports will be handled with priority and the reporter will be informed when the issue is addressed.
