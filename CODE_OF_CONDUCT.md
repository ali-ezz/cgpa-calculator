# Contributor Covenant Code of Conduct

## Our Pledge

We welcome contributors from all backgrounds and commit to providing a respectful, inclusive environment.

## Our Standards

Contributors should:

- Be respectful and considerate.
- Use constructive and professional language.
- Focus discussions on ideas and code.
- Avoid harassment, discrimination, or abusive behavior.

## Reporting

If you experience or witness unacceptable behavior, please report it to the repository maintainers.

## Enforcement

Maintainers may remove or edit content that violates this code of conduct.
